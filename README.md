# 일정 관리 API

##  개요
이 프로그램은 사용자의 일정을 체계적으로 관리할 수 있도록 설계된 일정 관리 API 서버입니다. 

일정 관리 API는 일정 관리를 위해 일정 생성, 전체 및 조건 조회, 삭제를 할 수 있습니다.

그리고 예외 처리 기능을 통해 안정성을 제공합니다.

>
> 
> **일정 관리 앱을 구현하기 위해 IntelliJ에서 Spring으로 만들어진 기본 프로젝트입니다.**

### 1. API 명세서

### (1) 일정 생성


| HTTP Method | URL               | 요청 Body 예시                                                                                                                                                 | 응답 Body 예시                                                                                                                                                                                                                     | 상태 코드        | 설명                   |
|-------------|-------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------|------------------------|
| POST        | /api/schedules     | ```json { "name": "John Doe", "pwd": "password123", "todo": "Complete project" } ```                                                                             | ```json { "id": 1, "name": "John Doe", "todo": "Complete project", "createdAt": "2024-12-10T12:00:00", "updatedAt": "2024-12-10T12:00:00" } ```                                                                                   | 201 Created       | 일정 생성 성공          |
|             |                   |                                                                                                                                                                |                                                                                                                                                                                                                                  | 400 Bad Request   | 요청 값 누락 시 오류     |



### (2) 전체 일정 조회


| HTTP Method | URL               | 요청 Body 예시                                                                                                                                                 | 응답 Body 예시                                                                                                                                                                                                                     | 상태 코드        | 설명                   |
|-------------|-------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------|------------------------|
| GET         | /api/schedules     | ```json { "name": "John Doe", "updatedAt": "2024-12-10" } ```                                                                                                 | ```json [ { "id": 1, "name": "John Doe", "todo": "Complete project", "createdAt": "2024-12-10T12:00:00", "updatedAt": "2024-12-10T12:00:00" } ] ```                                                                             | 200 OK           | 일정 조회 성공          |
|             |                   |                                                                                                                                                                |                                                                                                                                                                                                                                  | 204 No Content    | 데이터 없음            |


### (3) 선택 일정 조회


| HTTP Method | URL                       | 요청 Path 예시        | 응답 Body 예시                                                                                                                                                                                                                       | 상태 코드        | 설명                   |
|-------------|---------------------------|-----------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------|------------------------|
| GET         | /api/schedules/{id}        | /api/schedules/1      | ```json { "id": 1, "name": "John Doe", "todo": "Complete project", "createdAt": "2024-12-10T12:00:00", "updatedAt": "2024-12-10T12:00:00" } ```                                                                                   | 200 OK           | 일정 조회 성공          |
|             |                           |                       |                                                                                                                                                                                                                                      | 404 Not Found     | ID에 해당하는 일정 없음 |


### (4) 선택 일정 수정


| HTTP Method | URL                       | 요청 Path 예시        | 요청 Body 예시                                                                                                                                                                                                                                                                                      | 응답 Body 예시                                                                                                                                                                                                                       | 상태 코드        | 설명                   |
|-------------|---------------------------|-----------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------|------------------------|
| PUT         | /api/schedules/{id}        | /api/schedules/1      | ```json { "name": "John Doe", "todo": "Updated task", "pwd": "password123" } ```                                                                                                                                                                                                                     | ```json { "id": 1, "name": "John Doe", "todo": "Updated task", "createdAt": "2024-12-10T12:00:00", "updatedAt": "2024-12-10T12:30:00" } ```                                                                                      | 200 OK           | 일정 수정 성공          |
|             |                           |                       |                                                                                                                                                                                                                                                                                                    |                                                                                                                                                                                                                                     | 404 Not Found     | ID에 해당하는 일정 없음 |
|             |                           |                       |                                                                                                                                                                                                                                                                                                    |                                                                                                                                                                                                                                     | 401 Unauthorized  | 비밀번호 불일치        |


### (5) 선택 일정 삭제


| HTTP Method | URL                       | 요청 Path 예시        | 요청 Body 예시                                                                                                                                                                                                                                                                                      | 상태 코드        | 설명                   |
|-------------|---------------------------|-----------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------|------------------------|
| DELETE      | /api/schedules/{id}        | /api/schedules/1      | ```json { "pwd": "password123" } ```                                                                                                                                                                                                                                                                  | 204 No Content    | 일정 삭제 성공          |
|             |                           |                       |                                                                                                                                                                                                                                                                                                    | 404 Not Found     | ID에 해당하는 일정 없음 |
|             |                           |                       |                                                                                                                                                                                                                                                                                                    | 401 Unauthorized  | 비밀번호 불일치        |



#### * 에러 처리



| 상태 코드     | 오류 메시지             | 설명                                                                                   |
|---------------|-------------------------|----------------------------------------------------------------------------------------|
| 400 Bad Request | "Bad Request"           | 잘못된 요청 형식이나 값이 누락된 경우 발생                                             |
| 401 Unauthorized | "Unauthorized"          | 인증 실패(비밀번호 불일치 등)                                                           |
| 404 Not Found  | "Not Found"             | 요청한 ID에 해당하는 일정이 없을 때 발생                                               |
| 500 Internal Server Error | "Internal Server Error" | 서버 내부 오류 발생 시 발생                                                             |

### 2. ERD

| Column Name | Data Type           | Description               |
|-------------|--------------|---------------------------|
| id          | BIGINT        | Primary Key, 자동 증가   |
| name        | VARCHAR(255)      | 일정 이름                |
| pwd         | VARCHAR(255)        | 일정 이름                |
| todo        | VARCHAR(255)        | 해야 할 일               |
| created_at  | VARCHAR(255)  | 일정 생성 시간           |
| updated_at  | VARCHAR(255)    | 일정 수정 시간           |
